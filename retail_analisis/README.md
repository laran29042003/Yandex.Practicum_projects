## Проект: 
Ритейл. Оценка эффективности торговых точек.
## Статус:
Окончен
## Описание:
Оценка эффективности магазинов сети по продаже строительных материалов. Проведение исследовательского анализа данных.  Формулирование и проверка статистических гипотез.
## Используемые библиотеки:
pandas, matplotlib, plotly, seaborn, numpy, scipy
